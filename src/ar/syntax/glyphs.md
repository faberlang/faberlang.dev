تستخدم Faber الرموز حيث يكون الرمز بنيوياً. فيما يلي الجرد الكامل
لرموز المصدر التي يتعرف عليها المحلل المعجمي.

## تدفق القيم {#value-flow}

| الرمز | المعنى |
|-------|---------|
| `←` | ربط وقت التشغيل، وإعادة الإسناد، والتحوير |
| `→` | نوع إرجاع الدالة |
| `⇥` | مخرج بديل — نوع قناة الخطأ أو استرداد التحويل المضمن |
| `∴` | جسم "لذلك" المختصر (`إذا شرط ∴ أرجع س`) |

## شكل النوع {#type-shape}

| الرمز | المعنى |
|-------|---------|
| `∷` | إسناد النوع الثابت (تحويل وقت التصريف) |
| `↦` | تحويل وقت التشغيل (تحليل/إكراه قابل للفشل) |
| `∪` | نوع اتحاد مضمن (`T ∪ nihil`) |

## المقارنة {#comparison}

| الرمز | المعنى |
|-------|---------|
| `≡` `≠` | المساواة وعدم المساواة التامة |
| `<` `>` `≤` `≥` | الترتيب |
| `≈` `≉` | مساواة القيم العددية |

## المنطقية والبتية {#logical-and-bitwise}

| الرمز | المعنى |
|-------|---------|
| `∧` `∨` `⊻` `¬` | و، أو، xor، ليس |
| `⇐` `⇒` | إزاحة البت لليسار ولليمين |

## تحديثات الإسناد {#assignment-updates}

| الرمز | المعنى |
|-------|---------|
| `←` | معامل الإسناد الوحيد في التعابير |
| `⊕` `⊖` | عبارات الزيادة/النقصان اللاحقة (للأعداد القابلة للتحوير فقط) |

## التسلسل الاختياري وتأكيد عدم العدم {#optional-chaining-and-non-null-assertion}

| الرمز | المعنى |
|-------|---------|
| `?` `?.` `?[` `?(` | التسلسل الاختياري |
| `!` `!.` `![` `!(` | تأكيد عدم العدم |

## النطاقات {#ranges}

| الرمز | المعنى |
|-------|---------|
| `‥` | نقطة نهاية نطاق استبعادية |
| `…` | نقطة نهاية نطاق شمولية |

## محددات القيم الحرفية {#literal-delimiters}

| الرمز | النوع | الدور |
|-------|------|------|
| `'` | `ascii` | رموز آلية ثابتة |
| `"` | `textus` | سلسلة سطرية |
| `«` `»` | `textus` | سلسلة كتلية (علامات الاقتباس الفرنسية) |
| `` ` `` | `forma` | قالب مأسور |
| `|` | `octeti` | قيمة حرفية ست عشرية |
| `§` | ثقب القالب | عنصر نائب داخل `"…"`، `«…»`، `` `…` `` |

## علامات الترقيم {#punctuation}

| الرمز | الدور |
|-------|------|
| `(` `)` | التجميع والاستدعاء |
| `{` `}` | كتلة، أو قيمة حرفية للجنس، أو مستند JSON |
| `[` `]` | قيمة حرفية للقائمة والفهرسة |
| `.` | الوصول إلى العضو |
| `,` | الفاصل |
| `;` | فاصل العبارات |
| `:` | فاصل حقول JSON |
| `=` | شكل الحقل البنيوي (ليس إسناد وقت التشغيل) |
| `@` | علامة التعليق التوضيحي |
| `#` | تعليق سطري |
