Demo-tabs prototype v2

One component, two axes. Panels use real reader-pack vocabulary — every panel passes faber check --reader-locale, and all seven lower to the same program.

Toggle your OS light/dark mode to see both themes. Try ←/→ when a tab is focused. Disable JS (or set reduce-motion) to see the fallback.

1 · Tabbed variant (landing pages — example or locale axis)

Rests at soft elevation, lifts on hover/focus. Click a language.

f salve-munde.fab
English · la
functio salve(textus nomen) → textus {
    fixum textus msg ← "Salve, §!"(nomen)
    redde msg
}

incipit {
    fixum textus m ← salve("munde")
    nota m
}
ภาษาไทย · th-TH
ฟังก์ชัน salve(ข้อความ nomen) → ข้อความ {
    คงที่ ข้อความ msg ← "สวัสดี, §!"(nomen)
    คืน msg
}

เริ่ม {
    คงที่ ข้อความ m ← salve("โลก")
    แสดง m
}
简体中文 · zh-Hans
函数 salve(文本 nomen) → 文本 {
    常量 文本 msg ← "你好，§!"(nomen)
    返回 msg
}

入口 {
    常量 文本 m ← salve("世界")
    显示 m
}
繁體中文 · zh-Hant
函式 salve(文字 nomen) → 文字 {
    常數 文字 msg ← "你好，§!"(nomen)
    傳回 msg
}

入口 {
    常數 文字 m ← salve("世界")
    顯示 m
}
العربية · ar
دالة salve(نص nomen) → نص {
    ثابت نص msg ← "مرحبا، §!"(nomen)
    أعد msg
}

بداية {
    ثابت نص m ← salve("عالم")
    اعرض m
}
हिन्दी · hi
फलन salve(पाठ nomen) → पाठ {
    स्थिर पाठ msg ← "नमस्ते, §!"(nomen)
    लौटाओ msg
}

आरंभ {
    स्थिर पाठ m ← salve("दुनिया")
    दिखाओ m
}
Tiếng Việt · vi
hàm salve(vănbản nomen) → vănbản {
    hằng vănbản msg ← "Xin chào, §!"(nomen)
    trả msg
}

bắtđầu {
    hằng vănbản m ← salve("thế giới")
    in m
}
