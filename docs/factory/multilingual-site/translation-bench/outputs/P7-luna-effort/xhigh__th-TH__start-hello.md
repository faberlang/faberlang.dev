## สิ่งที่คุณเพิ่งใช้ {#what-you-used}

| แหล่งที่มา | ความหมาย |
|---|---|
| `functio salve(textus nomen) → textus` | ฟังก์ชันชื่อ `salve` พารามิเตอร์ที่ระบุชนิดไว้ก่อน และคืนค่าเป็นข้อความ |
| `fixum textus msg ← ...` | การผูกค่าที่เปลี่ยนแปลงไม่ได้ |
| `"Salve, §!"(nomen)` | สตริงรูปแบบที่มีการแทรกค่าที่แสดงผล |
| `redde msg` | ส่งค่ากลับ |
| `incipit` | จุดเริ่มต้นของแพ็กเกจ |
| `nota m` | พิมพ์โน้ตหรือค่าเอาต์พุต |

## การพิสูจน์โลแคล {#locale-proof}

โปรแกรมด้านบนคือการแสดงผลของตัวอ่านภาษาละตินแบบมาตรฐาน โลแคลของตัวอ่านสามารถแสดงโปรแกรมเชิงความหมายเดียวกันด้วยชุดคีย์เวิร์ดที่แตกต่างกัน โดยยังคงรูปอักขระและตัวระบุไว้ เริ่มจากการพิสูจน์ฉบับเต็มที่ [โลแคลของตัวอ่าน](/features/reader-locale.html) ก่อนเขียนแพ็กเกจที่ไม่ใช่ภาษาละติน
