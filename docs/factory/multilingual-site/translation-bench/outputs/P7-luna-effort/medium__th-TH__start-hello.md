## สิ่งที่คุณเพิ่งใช้ {#what-you-used}

| แหล่งที่มา | ความหมาย |
|---|---|
| `functio salve(textus nomen) → textus` | ฟังก์ชันชื่อ `salve` พารามิเตอร์แบบระบุชนิดก่อน และคืนค่าเป็นข้อความ |
| `fixum textus msg ← ...` | การผูกค่าที่เปลี่ยนแปลงไม่ได้ |
| `"Salve, §!"(nomen)` | สตริงรูปแบบที่มีการแทรกค่าที่จะแสดง |
| `redde msg` | คืนค่า |
| `incipit` | จุดเริ่มต้นแพ็กเกจ |
| `nota m` | พิมพ์ค่าบันทึก/เอาต์พุต |

## การพิสูจน์โลแคล {#locale-proof}

โปรแกรมด้านบนคือการแสดงผลของผู้อ่านภาษาละตินตามรูปแบบมาตรฐาน
โลแคลของผู้อ่านสามารถแสดงโปรแกรมเชิงความหมายเดียวกันด้วยชุดคีย์เวิร์ดที่แตกต่างกัน
โดยยังคงรักษาอักขระและตัวระบุไว้ เริ่มจากการอ่านข้อพิสูจน์ฉบับเต็มที่
[โลแคลของผู้อ่าน](/features/reader-locale.html) ก่อนเขียนแพ็กเกจที่ไม่ใช่ภาษาละติน
