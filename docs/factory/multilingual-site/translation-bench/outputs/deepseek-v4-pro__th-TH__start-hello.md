## สิ่งที่คุณเพิ่งใช้ {#what-you-used}

| Source | ความหมาย |
|---|---|
| `functio salve(textus nomen) → textus` | ฟังก์ชันชื่อ `salve` พารามิเตอร์แบบระบุประเภทก่อน ส่งคืนข้อความ |
| `fixum textus msg ← ...` | การผูกค่าที่เปลี่ยนแปลงไม่ได้ |
| `"Salve, §!"(nomen)` | สตริงรูปแบบที่มีการแทรกค่าแบบแสดงผล |
| `redde msg` | ส่งคืน |
| `incipit` | จุดเริ่มต้นของแพ็กเกจ |
| `nota m` | พิมพ์หมายเหตุ/ค่าเอาต์พุต |

## ข้อพิสูจน์ด้านโลแคล {#locale-proof}

โปรแกรมข้างต้นคือการเรนเดอร์ตามรูปแบบละตินที่เป็นมาตรฐาน โลแคลของตัวอ่านสามารถเรนเดอร์โปรแกรมที่มีความหมายเดียวกันด้วยชุดคำสำคัญที่แตกต่างกัน โดยยังคงรักษาสัญรูปและตัวระบุเดิมไว้ เริ่มต้นด้วยข้อพิสูจน์ฉบับเต็มที่ [Reader locale](/features/reader-locale.html) ก่อนเขียนแพ็กเกจที่ไม่ใช่ภาษาละติน
