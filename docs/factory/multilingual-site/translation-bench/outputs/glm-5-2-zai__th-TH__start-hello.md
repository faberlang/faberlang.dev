## สิ่งที่คุณเพิ่งใช้ {#what-you-used}

| ต้นทาง | ความหมาย |
|---|---|
| `functio salve(textus nomen) → textus` | ฟังก์ชันชื่อ `salve`, พารามิเตอร์แบบชนิดก่อน, คืนค่าข้อความ |
| `fixum textus msg ← ...` | การผูกที่เปลี่ยนแปลงไม่ได้ |
| `"Salve, §!"(nomen)` | สตริงรูปแบบที่มีการแทรกค่าสำหรับการแสดงผล |
| `redde msg` | การคืนค่า |
| `incipit` | จุดเริ่มต้นของแพ็กเกจ |
| `nota m` | พิมพ์บันทึก/ค่าผลลัพธ์ |

## ข้อพิสูจน์โลเคล {#locale-proof}

โปรแกรมข้างต้นคือการแสดงผลลัพธ์ภาษาละตินแบบอ่านได้ตามมาตรฐาน โลเคลผู้อ่านสามารถแสดงโปรแกรมเชิงความหมายเดียวกันด้วยชุดคำสำคัญที่แตกต่างกันได้ ในขณะที่ยังคงรักษาสัญลักษณ์และตัวระบุไว้ เริ่มจากข้อพิสูจน์ฉบับสมบูรณ์ที่ [โลเคลผู้อ่าน](/features/reader-locale.html) ก่อนเขียนแพ็กเกจที่ไม่ใช่ภาษาละติน
