## 系統提示摘要（讀者語言包）

發出繁體中文讀者語區的 Faber，在適合之處使用中文關鍵字來表示宣告、控制流程、迴圈、回傳、布林值、常見基礎型別以及清單／映射集合。保持 Faber 符文語法不變。僅使用繁體字（不使用簡體字）。

## 說明

將下列英文轉換為繁體中文（zh-Hant）的讀者語區。遵循以下規則：

- 僅轉換散文部分。請勿翻譯程式碼區塊。
- 原封不動保留 {#anchors}（標題錨點）。
- 保留 Markdown 結構（標題、段落、清單）。
- 保留表格形狀（欄數、對齊方式）。
- 保留連結。路徑可以保持絕對路徑（/start/…），因為產生器會自動加上語區路徑前綴。
- 區塊以 "<<<FENCE n>>>" 標記表示。請在輸出中原封不動保留這些標記。
- 回傳帶有標記的完整內文。
- 僅回傳已轉換的 Markdown 內文（無前置資訊、前言、附註或後記）。
- 全文使用繁體中文字元（繁體），絕不使用簡體。

## 你剛使用的內容 {#what-you-used}

| 原始碼 | 含義 |
|---|---|
| `functio salve(textus nomen) → textus` | 名為 `salve` 的函式，型別優先的參數，回傳文字 |
| `fixum textus msg ← ...` | 不可變繫結 |
| `"Salve, §!"(nomen)` | 帶顯示插值的格式化字串 |
| `redde msg` | 回傳 |
| `incipit` | 套件進入點 |
| `nota m` | 列印附註／輸出值 |

## 語區驗證 {#locale-proof}

上述程式是標準的拉丁文讀者呈現。不同的讀者語區可以在保留符文與識別碼的同時，使用不同的關鍵字集來呈現相同的語意程式。在撰寫非拉丁文套件之前，請先從[讀者語區](/features/reader-locale.html)的完整驗證開始。
