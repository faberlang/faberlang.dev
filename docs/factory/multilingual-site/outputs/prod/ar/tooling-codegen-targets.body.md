فابر لديها لغة واحدة وعقود تحويل برمجي متعددة. ليس بالضرورة أن تتحول كل ميزة إلى كل هدف. توثق هذه الصفحة الميزات التي يدعمها كل هدف، أو يُسقطها، أو يُحذر منها، أو يرفضها.

## أفعال السياسة {#policy-verbs}

| الفعل | المعنى |
|------|---------|
| **دعم** | يتحول بدلالات مقصودة |
| **إسقاط** | يتحقق من النوع؛ يُسقط منشئ الكود الدلالات الخاصة بالهدف |
| **تحذير** | فابر قانوني؛ لا تأثير أو سلوك متدهور على الهدف |
| **رفض** | يفشل التحقق أو الإخراج بتشخيص صريح |
| **تأجيل** | تحليل وربط ناجحان؛ التحويل غير مُنفَّذ لأي هدف |
| **محدود** | عقد مستقر مع بوابات مجموعة فرعية صريحة |
