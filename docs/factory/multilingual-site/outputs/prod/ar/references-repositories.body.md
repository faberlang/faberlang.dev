تُطوَّر Faber عبر مستودعات متعددة تحت مظلة منظمة `faberlang`.

## المستودعات العامة {#public-repositories}

| المستودع | الوصف |
|-----------|-------------|
| `faber` | واجهة سطر الأوامر للمستخدم: فحص، بناء، تشغيل، اختبار، تنسيق، شرح |
| `faber-runtime` | أنواع وقت التشغيل الأساسية (Valor، tensors، frames)؛ اسم الصندوق `faber` |
| `norma` | كود المكتبة القياسية (وحدات `norma:*`) |
| `triga` | مكتبة رسوميات وهندسة اختيارية |
| `cista` | مدير الحزم والمستودع (تجريبي) |
| `examples` | مجموعة اللغة، الأدوات الأساسية، AI Workbench، حزم قارئ اللغة المحلية |
| `faberlang.dev` | هذا الموقع |

## المستودعات الخاصة {#private-repositories}

| المستودع | الوصف |
|-----------|-------------|
| `radix` | المصرف: التحليل المعجمي، التحليل النحوي، التحليل الدلالي، HIR/MIR/AIR، التشخيص، توليد الكود |

## مستودعات المنصة المضيفة {#host-platform-repositories}

| المستودع | الوصف |
|-----------|-------------|
| `host-kernel-rs` | موجه رفيع: Frame، Conversation، توجيه بالبادئة، أخطاء منظمة |
| `host-native-rs` | ربط أصلي: عمال، خطاف register_providers |
| `host-providers-rs` | تطبيقات المزود: solum، processus، consolum، tempus، aleator، http |
