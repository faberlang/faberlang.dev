ดาวน์โหลดไฟล์เก็บถาวร Faber CLI ที่สร้างไว้ล่วงหน้า และดูทุกแท็กที่เผยแพร่แล้วในคลังแอสเซ็ต [faberlang/releases](https://github.com/faberlang/releases) หน้านี้ใช้สำหรับ **ติดตั้ง** compiler CLI ไม่ใช่สำหรับสร้าง compiler จากซอร์สโค้ด.

## เริ่มต้นใช้งาน {#getting-started}

| [ทัวร์สั้น ๆ](/start/) | รูปแบบของภาษาในเวลาไม่กี่นาที |

| [ตัวอย่าง](/start/examples.html) | แพ็กเกจตัวอย่างสาธารณะ |

คำแนะนำสำหรับการสร้างจากซอร์สโค้ดจะไม่แสดงลิงก์ไว้ที่นี่โดยตั้งใจ ใช้ไฟล์เก็บถาวรที่สร้างไว้ล่วงหน้า เว้นแต่คุณจะทำงานกับต้นไม้ซอร์สโค้ดของ compiler แบบส่วนตัว.
