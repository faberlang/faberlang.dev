Faber 使用分隔符语义 — 每种引号形式表示不同的源代码形态。它们之间不能互换。

## 字面量形式 {#literal-forms}

| 形式 | 类型 | 角色 |
|------|------|------|
| `'…'` | `ascii` | 固定机器令牌；无 `§`；无 `(…)` |
| `"…"` | `textus` | 短 Unicode 单行字符串；`(…)` 渲染 |
| `«…»` | `textus` | 块/多行 Unicode；`(…)` 渲染 |
| `` `…` `` | `forma` | 捕获模板；`(…)` 捕获 |
| `{ … }` | `json` | 编译期 JSON 文档 |
| `|…|` | `octeti` | 编译期十六进制字节 |
| `[ … ]` | `lista<T>` | Faber 列表字面量 |

Faber 通过字符串模板应用格式化文本：一个带有 `§` 占位符的 `"…"` 或 `«…»` 字面量，后接括号参数：

<<<FENCE 0>>>

关键规则：
- `§` (U+00A7) 是模板占位符
- 位置占位符：`§0`、`§1`、…… 用于显式排序
- 结尾的 `!` 选择显示格式化：`"Salve, §!"(nomen)`
- `(args)` 后缀是模板应用，不是函数调用
